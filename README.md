# Unik Food — PWA приложение

## Что это
Полноценное клиентское приложение (PWA) для Unik Food. Работает на телефоне как нативное приложение, устанавливается на домашний экран.

## Структура
- `index.html` — Каталог программ питания
- `lk.html` — Личный кабинет клиента
- `order.html` — Оформление заказа
- `history.html` — История заказов
- `payment.html` — Оплата
- `settings.html` — Настройки профиля
- `css/style.css` — Стили
- `js/app.js` — Логика приложения
- `js/api.js` — Интеграция с amoCRM + Pay Keeper
- `manifest.json` — PWA манифест
- `sw.js` — Service Worker (офлайн-режим)

## Бизнес-логика (уже встроена)
- Программы: Снижение, Баланс, Набор, Премиум, Detox
- Форматы: 5+5, 7+7, 10+10, 14+14
- Бонусы: +2 дня (5+5/7+7), +4 дня (10+10/14+14)
- Графики доставки по правилам Unik Food
- Заморозка до 10 дней
- Смена адреса до 22:00

## Развёртывание

### Вариант 1: GitHub Pages (бесплатно)
1. Создать репозиторий на GitHub
2. Загрузить все файлы в корень
3. Settings → Pages → main → Save
4. Приложение доступно по `https://yourname.github.io/repo/`

### Вариант 2: Vercel (бесплатно, быстрее)
1. Зарегистрироваться на vercel.com
2. Импортировать GitHub репозиторий
3. Готово — HTTPS + CDN

### Вариант 3: Свой сервер
1. Залить файлы на хостинг
2. Настроить HTTPS (обязательно для PWA)
3. Добавить домен в manifest.json

## Установка на телефон
1. Открыть ссылку в Chrome/Safari
2. Нажать «Поделиться» → «На экран Домой» (iOS) или «Установить приложение» (Android)
3. Иконка появится на рабочем столе — работает как нативное приложение

## Разработка
Для полноценного запуска нужен бэкенд (Node.js/Python) который будет:
- Проксировать запросы к amoCRM (CORS)
- Обрабатывать webhooks от Pay Keeper
- Отправлять Push-уведомления через Firebase
